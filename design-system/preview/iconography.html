Phosphor Icons · Regular
Sparsa uses Phosphor Icons at the Regular weight — a 1.5-unit visual stroke in a 32×32 frame. Filled paths using currentColor so any icon inherits the surrounding text color. Curated subset shipped in assets/icons/; the full 1500+ icon set lives on the CDN (see below).

Defaults: 20px in a 32×32 viewBox. Regular weight is the only weight Sparsa uses — do not mix in Thin / Bold / Fill / Duotone. Filled paths use currentColor so set the icon's color via the parent's color property, not the SVG itself.

CDN: for full icon catalog, load <link rel="stylesheet" href="https://unpkg.com/@phosphor-icons/web@2/src/regular/style.css"> then write <i class="ph ph-envelope"></i>.
